대회 진행방식

대회 진행방식은 아래와 같습니다. 반드시 규칙을 필히 숙지하고 참가부탁드립니다 :)

  • 1. 팀은 한팀당 최소 4명 최대 8명, 8팀으로 진행 (신청자 수에 따라 팀수는 변경될 수 있습니다) (미호서버를 최근 1시간 이상 플레이한 유저만 신청이 가능합니다)
  • 2. SIZE 2100의 맵에서 진행(발사장 포함)
  • 3. Warmup 15분 / Grace 50분 / Attack 40분 / Raid 80분
  • 3-1. *Warmup 시간은 움직일 수 없습니다, 팀원분들 모두 입장하셔야 하며 리더가 팀만들기를 눌러 클랜 생성 후 /clan invite steamid or 이름으로 초대합니다. *팀원은 /clan accept 클랜명을 입력해 가입합니다.
  • 3-2. *Grace 시간에 랜덤 리스폰되며 자원 파밍 후 베이스를건설해야 합니다.(서로 죽일 수 없음) *팀장은 반드시 Grace시간안에 /maintc 라는 명령어를 통해 메인 건차를 설정해야 함 *절대로 남의 집에 들어가면 안됩니다 고의(남의 집 근처에 10초이상 얼쩡거리는 행위 등)로 남의집에 들어가면 무조건 플레이어가 속한 클랜은 탈락합니다.(경고X 바로 탈락)
  • 3-3. Attack 시간에 벤딧이 가능함. (레이드 불가능)
  • 3-4. Raid 시간에 모든 제한이 풀리며 상대 클랜 베이스 레이드 가능. 메인 건차가 파괴되면 해당 클랜원들은 모두 아웃(서버에서 추방)되며 베이스가 사라짐.
  • 4. Raid시간이 종료시 Score에 표기된 등수에 따라 순위가 결정됨

※ 스코어경쟁 룰 ※

스코어 산정은 다음과 같이 산정됩니다.

  • 1) 플레이어 킬 - 0.5 Point
  • 2) 엘리트 상자 - 3 Point
  • 3) 치누크 상자 - 8 Point
  • 4) 브래들리 파괴 점수 - 10 Point ㄴ 브래들리상자 - 7 Point
  • 5) 헬기 파괴 점수 - 20 Point ㄴ 헬기상자 - 10 Point
  • 6) 건차 파괴 점수 - 100 point

대회 특징

대회 특징은 아래와 같습니다. 필히 숙지하고 참가부탁드립니다 :)

  • 1. 자원 10배 / 스폰시간 감소 / 잡템제거 / 파밍배율 5~10배 / 좋은 등급의 아이템 스폰
  • 2. Pvp(파랑), Raid(빨강) 발생시 지도에 마커로 표시됩니다.
  • 3. 건차는 클랜당 한 개의 TC만 설치 가능
  • 4. 터렛 3개 / 샷건트랩 2개 / 화염트랩 2개 / 샘사 1개 설치가능
  • 5. GRACE단계에서 고잉딥은 절대 불가(적발시 즉시 탈락+대회 블랙리스트)
  • 6. ATTACK단계부터 지도에 TC가 상점아이콘으로 표시됨
  • 7. 탱크, 카고, 공격헬기는 ATTACK단계서부터 스폰
  • 8. GRACE단계부터 채팅불가
  • 9. 소음기, 볼액, L96, 배율스코프 사용 불가
  • 10. 자동 권한공유(건차,터렛,문) / 모든 블프 개방
  • 11. 헬기 필드스폰 + 아웃포스트에서 구매 별도로 가능
  • 12. 즉시 제작 / 작업대 범위 확장
  • 13. 추위피해 X / 스폰시 풀컨디션으로 부활 / 방사능피해X(소리만 들림)
  • 14. 항상 낮 유지
  • 15. 카드키 없이도 문 열기 가능(북극연구기지 제외)
  • 16. 스킨박스 사용가능 명령어 : /sb
